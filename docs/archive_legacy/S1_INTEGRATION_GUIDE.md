### Example Conversion

**Input S1 Query**:
```
processName = "cmd.exe" AND
srcIp IN ("192.168.1.100", "192.168.1.101") AND
threatSeverity >= "high"
```

**Conversion Process**:

1. **Parse Query**:
   ```python
   conditions = [
       S1QueryCondition(
           field="processName",
           operator=S1Operator.EQUALS,
           value="cmd.exe"
       ),
       S1QueryCondition(
           field="srcIp",
           operator=S1Operator.IN,
           value=["192.168.1.100", "192.168.1.101"]
       ),
       S1QueryCondition(
           field="threatSeverity",
           operator=S1Operator.GREATER_EQUAL,
           value="high"
       )
   ]
   ```

2. **Map Fields** (via `s1_observo_mappings.py`):
   ```python
   processName → process.name
   srcIp → source.ip
   threatSeverity → event.severity
   ```

3. **Convert Operators**:
   ```python
   "=" → "=="
   "IN" → "in_list"
   ">=" → ">="
   ```

4. **Generate Lua**:
   ```lua
   function process(log)
       -- Filter: processName = "cmd.exe"
       if log["process"] and log["process"]["name"] == "cmd.exe" then
           -- Filter: srcIp IN (...)
           local src_ip = log["source"] and log["source"]["ip"]
           if src_ip and in_list(src_ip, {"192.168.1.100", "192.168.1.101"}) then
               -- Filter: threatSeverity >= "high"
               local severity = log["event"] and log["event"]["severity"]
               if severity and severity_level(severity) >= severity_level("high") then
                   return log
               end
           end
       end
       return nil
   end
   ```

5. **Output Observo Pipeline**:
   ```json
   {
       "name": "S1 Threat Detection",
       "type": "lua",
       "code": "...",
       "metadata": {
           "source": "SentinelOne",
           "converted_from": "processName = \"cmd.exe\" AND..."
       }
   }
   ```

### Example 2: Network Activity with Multiple Conditions

**S1 Query**:
```
srcIp IN ("192.168.1.0/24") AND dstPort = 443 AND protocol = "TCP"
```

**Observo Lua**:
```lua
function process(log)
    local src_ip = log["source"] and log["source"]["ip"]
    if src_ip and ip_in_cidr(src_ip, "192.168.1.0/24") then
        local dst_port = log["destination"] and log["destination"]["port"]
        if dst_port == 443 then
            local protocol = log["network"] and log["network"]["protocol"]
            if protocol and string.upper(protocol) == "TCP" then
                return log
            end
        end
    end
    return nil
end
```
